
一、移动端新适配vw
1、从屏幕左边到右边是100vw
2、font-size:13.3333333vw
3、推导：以750px设计稿为例：
	750px 100vw
	1rem = 100px
	7.5rem = 100vw
	1rem = 13.3333333vw

	其实就是将：1rem设置为100px，就可以免计算了。

四、修饰符.native
	组件绑定事件，如果不写修饰符.native就是监听组件内部发生的事情
	监听的是组件中点击了第一个元素的事件，写上修饰符.native

五、前进、后退 tab没有变化
用深度监控
watch:{
      $route: {
        handler(){
          this.selected = this.$route.name
        },
        immediate: true // 初始的是时候立即调用一次
      }
},
